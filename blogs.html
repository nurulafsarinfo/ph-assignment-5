Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:

  • innerHTML : Sets or retrives the html tags inside an element and it can be used to inject HTML dynamically.
  • innerText : It's retrives or sets the visible text inside an element and ignores hidden element. Does not parse HTML tag.
  • textContent:It's retrives or sets all text content inside an element and also including hidden element.Does not parse html tags

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling is a mechanism in javascript where an event start form the target element
and propagates up to its parent element in the DOM tree.
